# BookWorm Frontend

This repository is now **frontend-only**.

- Stack: React + Vite + Tailwind CSS
- Purpose: Book recommendation and reading tracker UI
- Backend: removed from this setup (integrate with a separate API service)

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
